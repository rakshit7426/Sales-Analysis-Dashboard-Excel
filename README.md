# Sales Analysis Dashboard

## Project Overview

This project is an interactive Sales Analysis Dashboard created in Microsoft Excel using a dataset of over 50,000 records. The dashboard helps analyze sales performance, track key business metrics, and identify sales trends through interactive visualizations.

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Filters & Slicers
- Conditional Formatting

## Dashboard Features

- Sales KPI Dashboard
- Interactive Slicers
- Pivot Charts
- Sales Trend Analysis
- Category-wise Analysis
- Region-wise Analysis

## Skills Demonstrated

- Data Analysis
- Dashboard Designing
- Data Visualization
- KPI Reporting
- Excel Reporting

## Dataset

- Dataset Size: 50,000+ Records
